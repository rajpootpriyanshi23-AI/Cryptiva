# Cryptiva
My  Cognifyz internship project's level2 task3
A Java-based File Encryption and Decryption application that securely encrypts and decrypts text files using a simple Caesar Cipher algorithm. The program reads data from an input file, processes it, and writes the result to a new output file.

## Features

- Encrypt text files
- Decrypt encrypted files
- Simple command-line interface
- File handling using BufferedReader and BufferedWriter
- Exception handling for invalid file paths

## Technologies Used

- Java
- File Handling
- String Manipulation
- Object-Oriented Programming (OOP)

## How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA or any Java IDE.
3. Compile and run `Cryptiva.java`.
4. Select:
   - **1** → Encrypt a file
   - **2** → Decrypt a file
5. Enter the input and output file names.
